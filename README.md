# Python 入門 & アルゴリズム学習プラットフォーム

このプロジェクトは、ブラウザだけで Python の基礎からアルゴリズム、データ構造までを体系的に学べるインタラクティブな学習サイトです。

## 主な特徴

- **環境構築不要**: [Pyodide](https://pyodide.org/) を使用し、ブラウザ上で直接 Python コードを実行できます。
- **インタラクティブな演習**: 各チャプターに紐付いた Jupyter Notebook (`.ipynb`) 形式の演習問題を、内蔵のエディタで解きながら学習を進められます。
- **豊富な可視化**: Matplotlib によるグラフ描写や画像処理の演習に対応しており、シミュレーション結果を視覚的に確認できます。
- **学習記録**: ローカルストレージを利用して、演習の進捗状況を記録できます。

## 学習カリキュラム

### 1. Python の基本
プログラミング未経験者でも、変数・関数・条件分岐・繰り返し・配列といった基本概念を順を追って習得できます。
- はじめての Python / 数と式 / 変数 / 関数
- ライブラリ / pip / 条件分岐 (if) / 繰り返し (for)
- 配列 / 多次元配列

### 2. アルゴリズムとデータ構造
数学的・物理的なシミュレーションを通じて、より高度なアルゴリズムとデータ構造を学びます。
- **物理シミュレーション**: 等速直線運動、放物運動、空気抵抗
- **数学的手法**: モンテカルロ法、再帰、連立一次方程式（ガウスの消去法）
- **基礎アルゴリズム**: 探索、ソート、計算量（オーダー）
- **応用**: 動的計画法 (DP)、画像処理、ライフゲーム、数値誤差
- **実践演習**: 素数判定、素因数分解、最大公約数 (GCD)

## 技術スタック

- **Frontend**: HTML5, CSS3, JavaScript
- **Python Runtime**: Pyodide (WebAssembly)
- **Editor**: Monaco Editor (VS Code と同等のエディタ体験)
- **Markdown Rendering**: Markdown-it, KaTeX (数式表示), Prism.js (シンタックスハイライト)
- **Styling**: GitHub Markdown CSS

## ディレクトリ構造

- `docs/`: 学習コンテンツ（MDX 形式）
- `static/`: 演習用ノートブック (`.ipynb`)、物理・数学ライブラリ、画像アセット
- `index.html`: メインのアプリケーション・エントリーポイント
