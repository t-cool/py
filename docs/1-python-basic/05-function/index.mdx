# <ruby>関数<rt>かんすう</rt></ruby>〜<ruby>魔法<rt>まほう</rt></ruby>の<ruby>機械<rt>きかい</rt></ruby>〜

ここでは、「<ruby>関数<rt>かんすう</rt></ruby>」という便利な<ruby>道具<rt>どうぐ</rt></ruby>について学びましょう。

## 関数って何ですか？

関数は、**「何かを入れると、何かをして出してくれる、<ruby>魔法<rt>まほう</rt></ruby>の<ruby>機械<rt>きかい</rt></ruby>」** だと思ってください。

例えば、「入れた数字を2乗（じじょう：同じ数字を2回かけること）にする機械」を作ってみましょう。
これがあれば、いちいち「2×2は…」と考えなくても、機械に「2」を入れるだけで「4」が出てくるのです。

<ViewSource path="/function/calc_x^2_revised.ipynb" />

Python では、関数をこのように作ります。

```python
def 機械の名前(中に入れるもの):
    return 出てくるもの
```

- `def`（デフ）は「これから機械を作るよ。」という<ruby>合図<rt>あいず</rt></ruby>です。
- `return`（リターン）は「これが答えだよ。」と外に出すことです。

:::caution スペースで区切りましょう
Python では、関数のなかみ（機械のなかでやる作業）を書くときに、かならず左側にスペースを入れて少し右にずらさないといけません。これを「インデント」と呼びます。
これを<ruby>忘<rt>わす</rt></ruby>れると、コンピュータが「どこまでが機械のなかみか分からないよ。」と<ruby>困<rt>こま</rt></ruby>ってエラーになってしまいます。

<ViewSource path="/function/indent.ipynb" />
:::

## 答えを出さない機械もありますか？

機械の中には、答え（数字）を出すのではなくて、「<ruby>画面<rt>がめん</rt></ruby>に文字を<ruby>表示<rt>ひょうじ</rt></ruby>する」だけのお<ruby>仕事<rt>しごと</rt></ruby>をするものもあります。

また、入れるもの（名前など）によって、言うことをかえる機械も作れます。

:::note メッセージシール `f-string`
上のプログラムのように、文字の中に変数をいれたいときは、 `f"こんにちは、{name}さん"` のように書くのがおすすめです。
最初に `f` と書いてから `{ }` で囲むと、その中に箱の中身がピタッと入ります。
:::
