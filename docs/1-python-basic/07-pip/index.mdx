# 新しい道具を手に入れよう！〜<ruby>pip<rt>ピップ</rt></ruby>〜

ここでは、新しい「<ruby>道具箱<rt>どうぐばこ</rt></ruby>」を手に入れる<ruby>方法<rt>ほうほう</rt></ruby>について<ruby>説明<rt>せつめい</rt></ruby>します。

## <ruby>pip<rt>ピップ</rt></ruby>って何ですか？

前のページで、Python にはいろいろな「<ruby>道具箱<rt>ライブラリ</rt></ruby>」があることをお話ししましたね。
でも、最初から使える道具箱は、実はほんの<ruby>一部<rt>いちぶ</rt></ruby>なのです。

世界中のプログラマーが作った「もっとすごい道具箱」を使いたいときは、インターネットからダウンロードしてくる必要があります。そのときに使うのが、 **<ruby>pip<rt>ピップ</rt></ruby>** というお<ruby>店<rt>おみせ</rt></ruby>のようなシステムです。

## pip の使い方

新しい道具箱を手に入れる（インストールする）には、このように書くだけで大丈夫です！

`!pip install 道具箱の名前`

これだけで、<ruby>自分<rt>じぶん</rt></ruby>のパソコンに新しい道具がやってきます。

ためしに、データ<ruby>分析<rt>ぶんせき</rt></ruby>が<ruby>得意<rt>とくい</rt></ruby>な `pandas`（パンダス）という有名な道具箱を手に入れてみましょう。

<ViewSource path="/pip/pandas.ipynb" />

:::note インストールは1回でOK
一度手に入れた道具箱は、ずっと皆さんのものになります！

:::
