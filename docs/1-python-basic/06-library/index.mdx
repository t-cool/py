# ライブラリ〜便利な<ruby>道具箱<rt>どうぐばこ</rt></ruby>〜

ここでは、「ライブラリ」という、プログラミングを助けてくれるすごい<ruby>道具箱<rt>どうぐばこ</rt></ruby>について学びましょう。

## ライブラリって何ですか？

Python には、<ruby>自分<rt>じぶん</rt></ruby>でいちから作らなくても、だれかが作ってくれた「便利な機械（関数）」がたくさん入った **「道具箱」** が<ruby>用意<rt>ようい</rt></ruby>されています。これを「ライブラリ」と呼びます。

例えば、<ruby>算数<rt>さんすう</rt></ruby>や<ruby>数学<rt>すうがく</rt></ruby>の難しい計算が<ruby>得意<rt>とくい</rt></ruby>な `math`（マス）という道具箱を使ってみましょう！

道具箱から道具を出すときは、最初に「この道具箱を使うよ！」と教えてあげる必要があります。

```python
import math
```

これだけで、`math` の中に入っているいろいろな計算ができるようになります。

## よく使う道具たち

`math` 道具箱には、このような道具が入っています（全部覚えなくて大丈夫ですよ！）。

| 道具の名前 | 何をしてくれる？ | 使い方の例 |
| ------------------ | ------------------------------------ | ------------------------------------------ |
| `math.pi`          | 円周率（えんしゅうりつ：3.14...）を教えてくれる | <ViewSource path="/library/pi.ipynb" />    |
| `math.ceil(x)`     | <ruby>小数点<rt>しょうすうてん</rt></ruby>を「切り上げ」する | <ViewSource path="/library/ceil.ipynb" />  |
| `math.floor(x)`    | 小数点を「切り捨て」する | <ViewSource path="/library/floor.ipynb" /> |
| `math.sqrt(x)`     | <ruby>平方根<rt>ルート</rt></ruby>を計算する | <ViewSource path="/library/sqrt.ipynb" />  |
