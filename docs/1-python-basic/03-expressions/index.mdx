:::tip スペースのなぞ
Python では、コードの間に「<ruby>半角<rt>はんかく</rt></ruby>スペース」が入っていても、<ruby>無視<rt>むし</rt></ruby>されることが多いです。ですから、読みやすくするためにスペースを入れても<ruby>大丈夫<rt>だいじょうぶ</rt></ruby>ですよ。

でも気をつけてください！ **「<ruby>全角<rt>ぜんかく</rt></ruby>スペース」が入っていると、エラーになって動かなくなってしまいます。** 日本語を入力したあとにスペースを入れるときは、特に<ruby>注意<rt>ちゅうい</rt></ruby>してくださいね！
:::

Python では、足し<ruby>算<rt>たしざん</rt></ruby>以外に、このような記号を使って計算ができます。

| 記号 | 何をする？ | 使い方の例 |
| ------ | ------ | -------------------------------------------------- |
| `+`    | たす   | <ViewSource path="/expressions/sum.ipynb" />       |
| `-`    | ひく   | <ViewSource path="/expressions/sub.ipynb" />       |
| `*`    | かける | <ViewSource path="/expressions/multi.ipynb" />     |
| `/`    | わる   | <ViewSource path="/expressions/div.ipynb" />       |
| `**`   | べき<ruby>乗<rt>じょう</rt></ruby> | <ViewSource path="/expressions/exp.ipynb" />       |
| `//`   | わり算の<ruby>商<rt>しょう</rt></ruby> | <ViewSource path="/expressions/quotient.ipynb" />  |
| `%`    | あまり | <ViewSource path="/expressions/remainder.ipynb" /> |

かっこの `(` と `)` を使うと、もっと難しい計算もできます。
