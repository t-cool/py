# もしも…だったら？〜<ruby>条件分岐<rt>じょうけんぶんき</rt></ruby>〜

## <ruby>正解<rt>せいかい</rt></ruby>か、間違いか？

プログラミングには、<ruby>数字<rt>すうじ</rt></ruby>や<ruby>文字<rt>もじ</rt></ruby>のほかに「正しい（`True`）」か「ちがう（`False`）」かを<ruby>表<rt>あらわ</rt></ruby>す<ruby>値<rt>あたい</rt></ruby>があります。

- **`True`（トゥルー）：** そのとおり。正解です。
- **`False`（フォルス）：** ちがいます。はずれです。

## 数字を比べてみよう

「5は3より大きいかな？」のように、数字を比べるときはこのような<ruby>記号<rt>きごう</rt></ruby>を使います。

| 記号 | 意味 | 使い方の例 |
