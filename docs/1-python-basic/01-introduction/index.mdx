# はじめに

このサイトは、Scratch（スクラッチ）などのブロックを組み合わせてプログラミングをしたことがある皆さんが、一歩進んで「テキストでプログラムを書く」ことに挑戦するための入門サイトです。

## ブロックからテキストへ

皆さんは Scratch などで、「10歩動かす」「ずっと繰り返す」といったブロックを組み合わせてプログラムを作ったことがあるかもしれません。直感的にプログラムが作れるのはとても便利ですよね。

しかし、世の中で動いている本格的なプログラムの多くは、文字（テキスト）で書かれています。

「テキストで書く」というのは、最初は難しく感じるかもしれません。でも、覚えるルールはそれほど多くありません。文字でプログラムが書けるようになると、より複雑なことを、より自由にコンピュータに命令できるようになります。

## なぜ Python なの？

このサイトでは、 **Python（パイソン）** というプログラミング言語を使います。
Python には、次のような特徴があります。

- **読みやすくて書きやすい：** 他の言語に比べて、英語に近い形で書けるので、初心者でも理解しやすいです。
- **サーバサイドで活躍：** Webサイトの裏側でデータを処理するなど、本格的なシステム開発に使われています。
- **人工知能（AI）に強い：** 今話題の AI やロボットの開発で、世界中で最も使われている言語の1つです。

## このサイトで学べること

このサイトでは、まず Python の基本的な書き方を学びます。そして、いくつかの「アルゴリズム（計算の手順）」の演習を通して、Python の扱い方に慣れていきます。

「アルゴリズム」を学ぶことは、プログラミングの基礎体力をつけることと同じです。ここで学んだことは、皆さんが AI を開発したり、新しいアプリを作ったりするときの強力な土台になります。

楽しみながら Python の世界に触れてみてください。

まずは基本から、一歩ずつ進めていきましょう！
