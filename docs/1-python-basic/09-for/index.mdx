## for 文のつかいかた

for 文は、このように書きます。

```python
for 名前 in range(はじまり, おわり):
    くりかえしたい作業
```

たとえば、「1 から 10 まで<ruby>表示<rt>ひょうじ</rt></ruby>してください」というプログラムはこうなります。

<ViewSource path="/for/print_1to10.ipynb" />

:::tip ちょっとだけ足していく `+=`
`sum += i` という書き方は、「いままでの `sum` の箱の中身に、 `i` を足して、また `sum` の箱に<ruby>戻<rt>もど</rt></ruby>してください！」という意味です。
これを「たし算しながら入れる」という<ruby>合図<rt>あいず</rt></ruby>だと思いましょう！
:::
