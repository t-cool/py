# もっと<ruby>複雑<rt>ふくざつ</rt></ruby>なデータ〜<ruby>多次元配列<rt>たじげんはいれつ</rt></ruby>〜

ここでは、配列をさらにパワーアップさせた「<ruby>多次元<rt>たじげん</rt></ruby>配列」について学びましょう。

## <ruby>二次元<rt>にじげん</rt></ruby>配列ってなに？

前のページでは「<ruby>算数<rt>さんすう</rt></ruby>の点数リスト」のように、ひとつの教科のリストを考えましたね。
でも、「算数、国語、英語」みたいに、たくさんの教科の点数をまとめたいときはどうすればよいでしょうか？

そんなときは、 **「<ruby>配列<rt>はいれつ</rt></ruby>のなかに、さらに配列を入れる」** ことができます。
これを「二次元配列」と呼びます。まるで **「あみだくじ」や「<ruby>表<rt>ひょう</rt></ruby>」** みたいですね。

| 名前 | 国語 | 数学 | 英語 |
| :--- | :--- | :--- | :--- |
| **Aさん** | 83 | 75 | 32 |
| **Bさん** | 73 | 53 | 84 |
| **Cさん** | 63 | 48 | 64 |

これをプログラムで書くと、このようになります。

## どうやって取り出すの？

二次元配列から数字を取り出すときは、「何行目の、何番目」というふうに<ruby>指定<rt>してい</rt></ruby>します。

```python
点数表 = [
    [83, 75, 32], # 0番目（Aさん）
    [73, 53, 84], # 1番目（Bさん）
    [63, 48, 64]  # 2番目（Cさん）
]

# Bさんの「数学（1番目）」の点数を取りたいとき
print(点数表[1][1]) # 53 が出てきます
```

ここでも **「0 からかぞえる」** のを忘れないでくださいね。
