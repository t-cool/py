# 数字あそびゲーム（FizzBuzz：フィズバズ）

## FizzBuzz ゲームをやってみよう。

<ruby>友達<rt>ともだち</rt></ruby>と順番に数字を言っていくゲーム「<ruby>FizzBuzz<rt>フィズバズ</rt></ruby>」を、プログラムで作りましょう。
この問題は、プログラミングができるかどうかのテストによく使われる、とても有名な問題です。

## ルールはこれだけ。

1から順番に数字を言っていきます。でも、こんなルールがあります。

1. **3で割り切れる** ときは、数字の代わりに「**Fizz**（フィズ）」と言います。
2. **5で割り切れる** ときは、数字の代わりに「**Buzz**（バズ）」と言います。
3. **3でも5でも割り切れる**（つまり15で割り切れる）ときは、代わりに「**FizzBuzz**（フィズバズ）」と言います。
4. それ以外のときは、そのままの数字を言います。

## どんな風に出るでしょうか？

正しくできると、こんな風になります。

```plaintext
1
2
Fizz
4
Buzz
Fizz
7
8
Fizz
Buzz
11
Fizz
13
14
FizzBuzz
```

## プログラムを作ってみよう

`if` 文（もし〜なら）を使って、<ruby>条件<rt>じょうけん</rt></ruby>をわけていくのがコツです。15の倍数（FizzBuzz）を一番先にチェックするのがポイントですよ。
