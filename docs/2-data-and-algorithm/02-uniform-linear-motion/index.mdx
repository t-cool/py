# <ruby>物体<rt>ぶったい</rt></ruby>のうごきをコンピュータでさいげんしてみましょう

# まっすぐすすむうごき（等速直線運動）

まっすぐ、ずっと同じ速さですすむ「<ruby>等速直線運動<rt>とうそくちょくせんうんどう</rt></ruby>」から考えてみましょう。

ボールが転がっているとき、ちょっとあとの<ruby>場所<rt>ばしょ</rt></ruby>はどうやって決まるでしょうか？

「ちょっとあとの場所 ＝ いまの場所 ＋ （速さ × ちょっとの時間）」

というふうに、いまの場所に「どれくらい進んだか」をたしていくことで、計算できるのです。

## プログラムを作ってみよう

コンピュータに、何度も何度もこの計算をさせることで、物体がどこにいくかを<ruby>予想<rt>よそう</rt></ruby>することができます。

<ViewSource path="/simulation/uniform_linear_motion.ipynb" />

:::note
むずかしい<ruby>数学<rt>すうがく</rt></ruby>を使うと、計算一回だけで答えを出すこともできますが、コンピュータは「なんども計算するのがとくい」ですから、上のような方法がよく使われます。
:::

プログラムができたら、実際にボールが動くようすを見てみましょう！

まずは、ボールが通った場所を全部メモするようにプログラムを書きかえます。
