# ボールを投げた時のうごき（放物運動）

ボールをななめに投げたときの動き「<ruby>放物運動<rt>ほうぶつうんどう</rt></ruby>」を考えてみましょう。

ボールを投げると、ふわっと山なりに飛んでいきますね。この動きは、２つの動きが<ruby>合体<rt>がったい</rt></ruby>したものなのです。

1. **よこ方向**：ずっと同じ速さで進みつづけます。
2. **たて方向**：<ruby>地球<rt>ちきゅう</rt></ruby>がひっぱる「<ruby>重力<rt>じゅうりょく</rt></ruby>」のせいで、だんだん下に落ちていきます。

この２つを合わせて計算すると、きれいなカーブがえがけます。
