# ぴったり重なる大きさを見つけよう（最大公約数）

## <ruby>最大公約数<rt>さいだいこうやくすう</rt></ruby>って？

２つの数字があったとき、そのどちらもぴったり割り切ることができる一番大きな数字を **「<ruby>最大公約数<rt>さいだいこうやくすう</rt></ruby>」** といいます。

英語では Greatest Common Divisor といって、よく **GCD** と<ruby>略<rt>りゃく</rt></ruby>されます。

## 2000年前から伝わるすごい技「ユークリッドの互除法」

むずかしい数式を使わなくても、**「あまり」** を使ってどんどん数字を小さくしていくと、答えがすぐに見つかるすごい方法があります。

これを「ユークリッドの<ruby>互除法<rt>ごじょほう</rt></ruby>」といいます。

1. 大きいほうの数字を、小さいほうの数字で割ります。
2. その「あまり」を使って、また同じことを繰り返します。
3. あまりが 0 になったときの数字が、<ruby>正解<rt>せいかい</rt></ruby>ですよ！
