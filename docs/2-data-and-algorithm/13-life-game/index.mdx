# コンピュータの中の小さな生き<ruby>物<rt>ライフゲーム</rt></ruby>

今回は、少し<ruby>複雑<rt>ふくざつ</rt></ruby>なプログラムに<ruby>挑戦<rt>ちょうせん</rt></ruby>してみましょう。「ライフゲーム」という不思議なシミュレーションを作ります。

## ライフゲームってなに？

ライフゲームは、1970年にイギリスの数学者が考えた、「<ruby>生命<rt>せいめい</rt></ruby>の<ruby>誕生<rt>たんじょう</rt></ruby>と死」をコンピュータの中でさいげんしたゲームです。

<ruby>碁盤<rt>ごばん</rt></ruby>の目のようなマス目に住む「<ruby>細胞<rt>さいぼう</rt></ruby>」たちが、まわりの友達の数によって生まれたり死んだりします。

## ライフゲームのルール

ルールはとてもシンプルです。それぞれのマスには「生きている（黒）」か「死んでいる（白）」の２つの状態があります。

次の<ruby>世代<rt>じだい</rt></ruby>でどうなるかは、まわりの８つのマスの<ruby>様子<rt>ようす</rt></ruby>で決まります。

1. **<ruby>誕生<rt>たんじょう</rt></ruby>**：死んでいるマスのまわりに、生きている友達が **ちょうど3人** いると、新しく命が生まれますよ。
2. **<ruby>生存<rt>せいぞん</rt></ruby>**：生きているマスのまわりに、生きている友達が **2人か3人** いると、そのまま生き残れます。
3. **<ruby>過疎<rt>かそ</rt></ruby>**：生きているマスのまわりに、生きている友達が **1人以下** しかいないと、さびしくて死んでしまうのです。
4. **<ruby>過密<rt>かみつ</rt></ruby>**：生きているマスのまわりに、生きている友達が **4人以上** もいると、苦しくて死んでしまうのです。

## プログラムを作ってみよう

コンピュータに、このルールを順番に教えてあげましょう。

1. <ruby>盤面<rt>ばんめん</rt></ruby>を 0 と 1 の数字で表します。
2. まわりの友達が何人いるか数える関数を作ります。
3. ルールに合わせて、次の世代の盤面を計算します。
