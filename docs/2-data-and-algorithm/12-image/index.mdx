# コンピュータが絵をかく仕組み（画像の表現）

コンピュータはどうやってきれいな絵や<ruby>写真<rt>しゃしん</rt></ruby>を<ruby>表示<rt>ひょうじ</rt></ruby>しているのか、のぞいてみましょう。

## 小さな点の集まり

コンピュータの<ruby>画面<rt>がめん</rt></ruby>を虫めがねで見ると、とても小さな四角い点々がいっぱい並んでいるのがわかります。この点の一つ一つを **「<ruby>画素<rt>がそ</rt></ruby>」** または「ピクセル」といいます。

## 白と黒のせかい

まずは、白と黒だけの絵を考えてみましょう。
コンピュータは「0」と「1」しかわからないから、こんなルールを決めます。

- **0** = 黒
- **1** = 白

これをたくさん並べると、文字や絵ができますよ。

<ViewSource path="/image/black_and_white.ipynb" />

## 色をつけるには？「光の<ruby>三原色<rt>さんげんしょく</rt></ruby>」

カラーの絵を描きたいときは、**赤・緑・青** の３つの色を混ぜてつくります。これを「光の三原色（RGB）」といいます。

- 赤をいっぱい、他は 0 → **真っ赤**
- 全部いっぱい混ぜる → **白**
- 全部 0 → **黒**

この３つの色の強さを数字で決めることで、どんな色でも作れるのです。

<ViewSource path="/image/color.ipynb" />

## きれいなグラデーションを作ろう

数字を少しずつ変えていくと、色がだんだん変わっていく「グラデーション」ができます。
絵の具をパレットの上で少しずつ混ぜていくのと<ruby>似<rt>に</rt></ruby>ていますね。

<ViewSource path="/image/black_to_red.ipynb" />

:::tip まめ知識
プリンターのインクは「シアン・マゼンタ・イエロー・ブラック（CMYK）」という色を使っています。画面の色（光）と、インクの色（絵の具）では、色のつくりかたが少しちがいます。
:::
