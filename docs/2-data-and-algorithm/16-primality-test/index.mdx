# <ruby>素数<rt>そすう</rt></ruby>を見つける名探偵

## <ruby>素数<rt>そすう</rt></ruby>ってなに？

「2, 3, 5, 7, 11, 13...」みたいに、**「1とその数字」でしか割り切れない特別な数字**のことを **「<ruby>素数<rt>そすう</rt></ruby>」** といいます。

たとえば、「6」は 2×3 で割り切れるから素数ではありませんが、「7」は 1×7 でしか表せないので素数なのです。

## コンピュータに調べさせてみましょう。

ある数字が素数かどうか、コンピュータに<ruby>名探偵<rt>めいたんてい</rt></ruby>になってもらって調べさせましょう。

どうやって調べるのでしょうか？
1. その数字より小さい数で、順番に割ってみます。
2. もし一度も割り切れなかったら、それは素数ですね。

## もっと速く調べるワザ

実は、全部の数字で割ってみなくても、その数字の「ルート（平方根）」まで調べれば十分なのです。

たとえば 100 が素数か調べるとき、10×10＝100 ですから、10まで調べれば答えがわかります。これで計算がとっても速くなるのですよ。
