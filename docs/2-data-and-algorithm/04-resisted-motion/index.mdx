# <ruby>風<rt>かぜ</rt></ruby>のじゃまがある時のうごき（空気抵抗）

<ruby>現実<rt>げんじつ</rt></ruby>の世界では、走ったり<ruby>自転車<rt>じてんしゃ</rt></ruby>に乗ったりすると、風を感じますよね。
物体が動くとき、空気は「うごきをじゃまする」のです。これを「<ruby>空気抵抗<rt>くうきていこう</rt></ruby>」といいます。

速く動けば動くほど、空気のじゃまする力も大きくなります。
