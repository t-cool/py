# 数字をバラバラに分解してみよう（素因数分解）

## 数字の「組み立て図」

どんなに大きな数字も、実は **「<ruby>素数<rt>そすう</rt></ruby>」** のかけ算だけで作ることができます。

たとえば、「12」をバラバラにしてみると……
$12 = 2 \times 2 \times 3$

というふうに、2と3という素数だけでできているのがわかりますね。これを **「<ruby>素因数分解<rt>そいんすうぶんかい</rt></ruby>」** とよびます。
