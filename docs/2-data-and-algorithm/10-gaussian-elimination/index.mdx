# ヒントを組み合わせて正解を見つけるパズル（連立方程式）

## これ、いくらかわかりますか？

算数のパズルをやってみましょう！

- ヒント1：リンゴ1個とミカン2個で 300円
- ヒント2：リンゴ2個とミカン1個で 330円

リンゴとミカンは、それぞれ1個いくらでしょうか？
こんなふうに、いくつかのヒント（式）を組み合わせて、わからない数字を当てる問題を **「<ruby>連立方程式<rt>れんりつほうていしき</rt></ruby>」** といいます。

## ガウスの<ruby>消去法<rt>しょうきょほう</rt></ruby>

ヒントが2つならかんたんですが、ヒントが100個もあったら、人間は<ruby>困<rt>こま</rt></ruby>ってしまいますね。
でも、コンピュータに **「ガウスの消去法」** という<ruby>手順<rt>てじゅん</rt></ruby>を教えてあげれば、どんなにむずかしいパズルでもスラスラ解いてくれるのです！

### どうやって解くの？

手順は大きくわけて２つです。

1. **お<ruby>掃除<rt>そうじ</rt></ruby>：** ヒントを組み合わせて、式をシンプルにしていきます。たとえば「リンゴの<ruby>値段<rt>ねだん</rt></ruby>だけ」がわかるように、他のものを消していくのです。
2. **あてはめ：** ひとつ答えがわかったら、それを他のヒントにも入れていって、全部の答えを<ruby>導<rt>みちび</rt></ruby>き出します。

:::info
数学では、数字をきれいに並べた「<ruby>行列<rt>ぎょうれつ</rt></ruby>」という道具を使って計算します。ちょっとむずかしそうに見えるけど、やっていることは「パズルのお掃除」なのです。
:::
