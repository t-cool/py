# たくさん<ruby>実験<rt>じっけん</rt></ruby>して正解を見つける方法（モンテカルロ法）

ここでは、「モンテカルロ<ruby>法<rt>ほう</rt></ruby>」という、ちょっと変わった計算の方法を学んでいきましょう。

## モンテカルロ法ってなに？

世の中には、<ruby>数学<rt>すうがく</rt></ruby>の計算だけでは答えが出せない、むずかしい問題がたくさんあります。

そんなとき、「バラバラな数字（乱数）」を使って、何度も何度も<ruby>実験<rt>じっけん</rt></ruby>してみることで、だいたいの答えを見つける方法があります。これを **モンテカルロ法** とよぶのです。

カジノで有名な「モンテカルロ」という名前がついているのは、ギャンブルみたいに<ruby>運<rt>うん</rt></ruby>にまかせて数字を決めるからなのです。

## 実験2：ダーツで「<ruby>円周率<rt>えんしゅうりつ</rt></ruby>」を当てましょう。

算数でならう「円周率（3.14...）」も、モンテカルロ法でもとめることができます。
